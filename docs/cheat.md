# TMUX CHEAT SHEET - PREFIX: Ctrl+b

## SESSIONS

d    Detach
$    Rename
s    List all
L    Last session (toggle)
(    Previous session
)    Next session

## WINDOWS

c    Create
,    Rename
&    Kill
w    List all
f    Find
n    Next
p    Previous
l    Last window (toggle)
0-9  Go to number

## PANES

-      Split horizontal
|      Split vertical
←↑→↓   Switch pane
C-←↑→↓ Resize pane
z      Zoom toggle
x      Kill
{      Swap left
}      Swap right
q      Show numbers
;      Last pane (toggle)

## COPY MODE (PREFIX + [)

Space  Start selection
Enter  Copy selection
/      Search forward
?      Search backward
n      Next match
N      Previous match

## OTHER

r    Reload config
/    This cheat sheet
T    Toggle light/dark theme
?    List all keybindings
:    Command prompt
